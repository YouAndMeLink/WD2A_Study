## 브랜치란?
- 코드를 통째로 복사하고 메인 코드와는 상관 없이 독립적으로 개발을 진행 할 수 있는데 이렇게 개별적으로 개발하는 개념
- 여러 개발자들이 동시에 다양한 작업을 할 수 있게 만들어 주는 기능
- 즉, 각각의 브랜치는 다른 브랜치에 영향을 받지 않기 때문에 여러 작업을 동시에 진행이 가능하다!
